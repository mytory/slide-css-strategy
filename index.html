안형우 (노동자 연대)

우리가 CSS를 배운 방식

.issue { ... }
                .issue h2 { ... }
                #intro h1 { ... }
                #intro .article h1 { ... }
이런 방식은 두 가지 권장 사항에 기반. 1. 시맨틱 - 콘텐츠의 본질을 반영하는 이름을 사용하라. 2. 계층적 선택자 - CSS의 C에 해당.

권장사항이 우리 목을 조르고 있다
